System settings

Enable Download Notifications

Show system tray notifications when new images are downloaded during automation or manual runs.

Enable Notification Sound

Play system alert sound when tray notifications are shown.

Close to System Tray

When enabled, clicking the window close button (X) will minimize the app to the system tray instead of quitting. The taskbar icon will disappear and the app continues running in the background.

Hide CLI Window on Startup

When enabled, launching the application will hide the command prompt/terminal window. (Requires starting via run.vbs)

Notification Click Action

Configure the action when clicking the system tray popup notification: open in default viewer or show download folder.

Phrases used to detect when Gemini refuses a request. The engine checks if any of these strings appear in the response text (case-insensitive). Edit to keep up with Gemini's changing refusal phrasing.
